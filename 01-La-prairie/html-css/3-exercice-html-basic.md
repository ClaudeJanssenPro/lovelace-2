# Bonjour, toi!

## Objectif de la séquence

- Comprendre le prototypage et faire un premier prototype dans Balsamiq Mockup
- Créer un site fictif d'une page

## Instructions de cet exercice

- Créer une maquette avec un logiciel de maquettage (comme Balsamiq Mockup) dans l'objectif de faire un site internet OnePage pour une opération Caritative au choix (par [ici](http://www.dons-legs.be/v2/listing-des-associations-solidarite-internationale-18649/) par exemple)
- Utiliser le fichier `index.html` pour exprimer tout ça (tu peux utiliser du contenu de site existant pour les textes et pour les images google avec les outils de recherche te permettra de trouver des images libre de droits)
- Dans ces paragraphes, mettre en évidence un ou plusieurs mots importants en utilisant le gras. Si tu as des mots dans une autre langue, utilise alors l'italique pour l'indiquer.
- Inclure une citation que tu aimes et qui te représente bien. (Tu peux aller voir sur [thatsthespir.it](https://thatsthespir.it)).
- Citer la source de ta citation.
- Faire en sorte que si l'on clique dessus, on y arrive dans une nouvelle fenêtre.
- Faire un sous-titre "D'autres associations" suivie d'une liste de liens vers des associations similaires.
- publie cela comme une GitHub Page

## Bonus

- Insérer une vidéo prise sur Youtube.
